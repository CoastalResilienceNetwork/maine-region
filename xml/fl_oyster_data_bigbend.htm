Process_Step:
Process_Description:
Oyster data was stereoscopically photo interpreted from 1:24000 scale natural color aerial photography. The FDOT Florida Land Use, Cover and Forms Classification System (FLUCCS) was used, in modified form, to code features within the 2001 Oyster study area boundary. Minimum mapping unit (mmu) for all classes was 0.5 acres. A Photo-interpretation Key was developed to aid in the classification of collected data. Ground truthing was performed during the photo-interpretation phase to ensure classification accuracy and consistency of PI. Aerial photography was aero-triangulated utilizing GCP points collected from DOQQs by Baymont using Arcview software. An Arcview shapefile was created documenting the GPS ground control points that were used and the control points collected from DOQQs. Digital compilation of the new baseline maps was carried out by photo-interpreters experienced in benthic habitat delineation, using a Zeiss P3 analytical stereoplotter. Digital files were created in Microstation design file format (.dgn). Microstation .dgn files were imported to ARC/Info to perform automated validation routines. Quality control checks for digital data consisted of unlabeled, multi-labeled and contiguous polygon errors as well as dangles and sliver polygons. Polygon topology was created and label errors and node errors were run in ARC/Info. Coverage characteristics were adhered too, as specified by the contract (tolerances, projection definition, PAT items etc.). Acreage calculations were derived from the ARC/Info coverages and delivered in EXCEL spreadsheet format.

Attribute_Value_Accuracy_Explanation:
(Overall accuracy = Number of corrected samples/total number of samples) The results of the accuracy assessment testing indicate that for the 20 points visited in the field, 20 of those points matched between the map and the fieldwork.
Once Baymont had completed fieldwork, photointerpretation, mapping and GIS within the Suwannee Estuary, an independent accuracy assessment of the resultant map data was conducted by Scheda Ecological Associates, Inc. (SEA) under subcontract with Baymont in order to ensure that the data adhered to the accuracy specifications required by the SRWMD. Scheda was tasked with visiting each sample point using watercraft and Differential Global Positioning System (DGPS) equipment. Using the sample points as waypoints in the DGPS Scheda visited each point and recorded the cover type for each into an Excel spreadsheet. These results were then placed into a classification distribution matrix of matches and mismatches. Random coordinate sampling and a comparison of field derived classification with mapped classification was used for assessing accuracy of the oyster mapping produced for the Suwannee Estuary. The stratified sample point coverage was created so that the oyster category would be sampled with a proportionally larger number of sample points. Both oyster and non-oyster sample points were used so that the entire domain was tested. Twenty points were selected and verified in the field for the assessment. The original contract specification called for an overall accuracy of 90%.
