Welcome to the Penobscot Habitat Blueprint Barrier Pioritization Tool

This tool can be used to help identify which barriers to aquatic connectivity would provide the greatest ecological gain if removed. Separate scenarios can be run using consensus weights developed by the Penobscot Habitat Blueprint working group for both resident and diadromous fish species. Additionally, users can define custom scenarios by applying relative weights to the metrics of their choice to reflect their objectives.

This map and prioritization tool are designed to be screening-level tools that can be used to help investigate potential fish passage projects in the context of many ecological factors. Results do not incorporate important social, economic, or feasibility factors and are not intended to be a replacement for site-specific knowledge nor a prescription for on-the-ground action.
